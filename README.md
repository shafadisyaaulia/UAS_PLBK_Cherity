#  SOLVIA - Chemistry Virtual Lab

Platform praktikum kimia virtual dengan **Computer Vision & AI**.

##  Quick Start

\\\ash
# Install
npm install
cd api && pip install -r requirements.txt

# Run Backend
cd api && python main.py  # Port 8000

# Run Frontend
npm run dev  # Port 3001
\\\

##  Fitur

- **Drag & Drop Mode**: Klik kimia untuk praktikum
- **Computer Vision Mode**: Kontrol dengan gesture tangan
- **pH Meter Real-time**: Perhitungan otomatis
- **Safety Detection**: AI monitoring
- **Modul Lengkap**: SD, SMP, SMA

##  Deploy ke Vercel

\\\ash
npm i -g vercel
vercel --prod
\\\

Backend deploy terpisah (Railway/Render).
